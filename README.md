# Run Instructions
Python version: 3.7.3

1. Install requirements: 
  ```bash
  pip install -r requirements
  ```
2. To run application: 
  ```bash
  python main.py
  ```

# To Run Tests
In app directory (same directory as main.py), run the following command: 
```bash
python -m pytest . 
```

# Notes
- Cache filepaths are specified in ./config.py
